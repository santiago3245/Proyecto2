# Proveedor Service

Microservicio para la gestión de proveedores en el sistema de inventario.

## Endpoints

- GET /proveedores - Obtener todos los proveedores
- GET /proveedores/{id} - Obtener proveedor por ID
- GET /proveedores/ruc/{ruc} - Obtener proveedor por RUC
- GET /proveedores/pais/{pais} - Obtener proveedores por país
- GET /proveedores/estado/{estado} - Obtener proveedores por estado
- POST /proveedores - Crear nuevo proveedor
- PUT /proveedores/{id} - Actualizar proveedor
- DELETE /proveedores/{id} - Eliminar proveedor
